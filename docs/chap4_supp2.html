更新した .wasm ファイルがWebブラウザに読み込まれない

原因：Webブラウザがキャッシュされた .wasm ファイルを使用するため。

対策：WebブラウザからのHTMLファイルのリロードには3種類あります。
以下は、Windows 上での Google Chrome と Microsoft Edge の場合です。

  1. ブラウザの再読込ボタン（Ctrl+R、または F5）
  2. ハードリフレッシュ（Shift+Ctrl+R、または Shift+F5）
  3. Ctrl+Shift+Iで開発者ツールを開いた状態で、ブラウザの再読込ボタンの右クリック→"キャッシュをクリアしてハードリフレッシュ"をクリック
上記1と2ではHTMLから間接的に読み込まれるwasmファイルは更新されませんが、3の方法では更新されます。

この他に、開発者ツールを開いている間のみ、キャッシュを無効化する方法もあります。
それは、Ctrl+Shift+Iで開発者ツールを開き、ツール上部の「ネットワーク」タブ→「キャッシュを無効にする」にチェックを入れます。
