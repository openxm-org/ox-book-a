Webサーバを介さずに（.jsファイルを読み込む）HTMLファイルをローカルで実行する方法

原因：.jsファイルを読み込むHTMLファイルをローカルで実行できないのは、Webブラウザのセキュリティ設定が原因です。 デフォルトでは、どのWebブラウザもHTMLファイルからローカルの.jsファイルを読み込むのを制限しています。

対策：Webブラウザのセキュリティ設定を変更すればこの制限は回避可能です。 ただし、危険性を十分理解した上で設定して下さい。
以下は、Google Chrome の場合です。

  1. プログラムメニューから Google Chrome を探して右クリック→More→Open file location
  2. Google Chrome のショートカットをデスクトップにコピー
  3. コピーしたショートカットを右クリック→プロパティを開く
  4. リンク先に入力されているパスの後に --allow-file-access-from-file を入力し、OKをクリック
  5. 次に Chrome を全て閉じる。※タスクマネージャやProcess Explorerなどで完全に落ちていることを確認
  6. Chrome が閉じたことが確認できたら、変更したショートカットで Chrome を起動
  7. Chrome のアドレスバーに chrome://version と入力し、オプションが適用されているか確認
以上で、.jsファイルを読みこむHTMLファイルを Chrome にドラッグ＆ドロップすることで実行できるようになります。
